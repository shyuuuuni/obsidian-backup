
# 유형

- 키워드에 대한 질문 답하기
- 화면에 질문이 뜸 (토픽에 대한 지문) - 독해 능력 중요
- 출제 빈도 높은 거 위주로 공부 + 템플릿 굳이 필요 없음

### 상황 유형

- 인터뷰 상황 (~를 인터뷰 하기로 했다)
- 전화 통화 상황 (~와 통화를 하고 있다)

### 5번

- 키워드 확인
- imagine ~ about 키워드

### 6번

- 의문사(1~2개) + 이유
- 만능공식 + 만능 문장 50 활용
### 7번

- 의견/선호도/장단점/옵션선택 추천
- 만능 문장 50 + 파트5 만능문장 + 3가지 답변 공식
